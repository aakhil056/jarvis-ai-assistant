# 🤖 Jarvis AI Assistant

A powerful AI assistant that can chat, understand voice, process images, and respond in real-time — inspired by ChatGPT and modern AI agents.

---

## 🚀 Features

- 🎤 Voice input (speech-to-text)
- 🔊 Voice response (text-to-speech)
- ⚡ Real-time streaming responses (like ChatGPT)
- 🧠 Conversation memory
- 🖼️ Image text extraction (OCR)
- 🤖 AI responses using Ollama (Llama3)
- 📱 Telegram bot integration
- 🎯 Wake word support ("Hey Jarvis")
- 🛑 Interrupt speech ("stop")

---

## 🛠️ Tech Stack

- Python
- Vosk
- pyttsx3
- Ollama (Llama3)
- Telegram Bot API
- pytesseract
- PIL
